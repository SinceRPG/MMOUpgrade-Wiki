MMOItems Upgrade System

Configure MMOUpgrade from A to Z.

MMOUpgrade adds a configurable GUI for upgrading MMOItems gear with upgrade stones, lucky charms, protect charms, level-based success rates, stat modifiers, Premium enchantment modifiers, per-item custom stats, and Paper/Folia-safe behavior.
